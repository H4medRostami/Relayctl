
# relayctl

A lightweight Bash utility to control your Uswitch 2-relay device with smart ON/OFF logic and live status monitoring.


## Features

- Smart `on` and `off` commands (checks current status before acting)
- Force `toggle` support
- Live monitoring with auto-refresh
- Beautiful colored terminal output with fancy ASCII banner
- Pure Bash — no external dependencies

---

## Usage

```bash
./relayctl.bash <command> [argument]
